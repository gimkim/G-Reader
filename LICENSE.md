# Fast Reader/Viewer license information

Fast Reader/Viewer includes third-party components that remain under their own licenses.
This file records the distribution terms for the PDF renderer bundled with the
application; it does not replace the licenses of the other NuGet dependencies.

## PDFium renderer

Fast Reader/Viewer uses PDFiumCore and the packaged PDFium native binary under their
permissive terms. Distributions containing those components must retain:

- `licenses/APACHE-2.0-PDFIUMCORE.md`
- `licenses/PDFIUM-BSD.md`

See [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md) for component versions and
links to their upstream projects.
